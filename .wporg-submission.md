## Plugin Description

AI Widget by Workfluz is an AI-powered chat and voice widget for WordPress that integrates with OpenAI, VAPI, and ElevenLabs to provide intelligent text and voice interactions for website visitors.

## External Services Used

### 1. OpenAI API
- **Purpose**: AI chat completions
- **Website**: https://api.openai.com
- **Privacy Policy**: https://openai.com/privacy
- **Terms of Service**: https://openai.com/terms
- **User Data**: Messages and conversation context sent to OpenAI servers
- **API Key Required**: Yes (user must provide their own)

### 2. VAPI SDK
- **Purpose**: Real-time voice call functionality
- **Website**: https://vapi.ai
- **Privacy Policy**: https://vapi.ai/privacy
- **Terms of Service**: https://vapi.ai/terms
- **User Data**: Voice audio and call metadata sent to VAPI servers
- **API Key Required**: Yes (user must provide their own)
- **CDN Used**: https://cdn.jsdelivr.net/gh/VapiAI/ (for loading SDK)

### 3. ElevenLabs API (Optional)
- **Purpose**: Text-to-speech synthesis
- **Website**: https://elevenlabs.io
- **Privacy Policy**: https://elevenlabs.io/privacy
- **Terms of Service**: https://elevenlabs.io/terms
- **User Data**: Text sent for voice synthesis
- **API Key Required**: Yes (user must provide their own, optional service)

## Data Collection & Privacy

**What We Collect Locally**:
- Session IDs (stored in browser cookies)
- Conversation history (stored in WordPress database)
- Usage statistics (message count, voice minutes)
- Installation configuration

**What We DON'T Collect**:
- Personal information (unless provided by users in conversations)
- Payment information (not implemented yet)
- Analytics or tracking data sent to external servers

**External Data Processing**:
- User messages sent to OpenAI for AI responses
- Voice audio sent to VAPI for voice processing
- Text sent to ElevenLabs (if enabled) for voice synthesis

**Compliance**:
- Plugin clearly discloses all external services in readme.txt
- Users must provide their own API keys (no automatic data sharing)
- Privacy policy section included in readme.txt
- Admin notices inform users about external services
